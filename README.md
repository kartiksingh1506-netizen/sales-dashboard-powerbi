# 📊 Sales Performance Dashboard

## 📌 Project Overview
This project is a Sales Performance Dashboard built using Power BI and PostgreSQL.  
It provides insights into sales trends, top-performing products, and regional performance.

---

## 🛠️ Tools & Technologies
- Power BI  
- PostgreSQL  
- SQL  
- Excel  

---

## 📊 Key Features
- Total Sales, Orders, and Profit analysis  
- Top 5 Products and Customers identification  
- Sales trend over time (monthly analysis)  
- Region-wise performance analysis  
- Interactive slicers for dynamic filtering
